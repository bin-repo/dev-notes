# DOM

文档对象模型（Document Object Model）

## 预备知识

- HTML
- CSS
- JavaScript

## W3C（万维网联盟）标准

- 定义了访问 HTML 和 XML 文档的标准

## W3C DOM 标准被分为3个部分

- `核心 DOM` 针对任何结构化文档的标准模型
- `XML DOM` 针对 XML 文档的标准模型
- `HTML DOM` 针对 HTML 文档的标准模型

## 概念

- 根节点（root node）
- 父节点（parent node）
- 子节点（child node）
- 同胞节点（sibling node）
- 属性节点（attribute）
- 属性取值（value）
- 文本节点（text）
